Zappe

Zappe est un ensemble de micro services permettant de soulager certains stresses aux développeurs sur des projets frontend par rapport à la recherche de lien d'image à intégrer directement ou à récupérer une liste d'images, etc...

Get one picture

URL: https://zappe.herokuapp.com/api/public/pic/{query}/{index}

PARAMS

{query} TYPE: string EXEMPLE: mia khalifa

{index} TYPE: int EXEMPLE: 0

RESULT TYPE: link (redirigé)

Get all pictures

URL: https://zappe.herokuapp.com/api/public/pics/{query}/{page}

PARAMS

{query} TYPE: string EXEMPLE: mia khalifa

{limit} TYPE: int EXEMPLE: 10

RESULT TYPE: array (de taille {limit})
