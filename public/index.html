Zappe

Zappe est un ensemble de micro services permettant de soulager certaines stresses aux développeurs sur des projets frontend par raport à la recherche de lien d'image à intégrer directement ou récupérer la une liste d'image, etc...

Get one picture

URL: https://zappe.herokuapp.com/api/public/pic/{name}/{index}

PARAMS

{name}
TYPE: string EXEMPLE: iamkepo

{index}
TYPE: int EXEMPLE: 0 à 9

RESULT
TYPE: link

Get all pictures

URL: https://zappe.herokuapp.com/api/public/pics/{name}/{page}

PARAMS

{name}
TYPE: string EXEMPLE: iamkepo

{page}
TYPE: int EXEMPLE: 1

RESULT
TYPE: array de taille 10
